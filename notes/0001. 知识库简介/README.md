## 1. 📝 概述

- **`TNotes.introduction`**
  - `TNotes.introduction` 是一个基于 `TNotes` 实现的一个开源的知识库，主要用于介绍 `TNotes` 的相关内容，可以理解为 `TNotes` 的使用说明文档。
- 在这个知识库下，主要会介绍 `TNotes` 的一些功能以及其使用说明，比如：
  - 自定义知识库封面
  - 自定义组件
  - markdown 增强
  - 本地搜索配置
  - 配置文件说明
  - ……
